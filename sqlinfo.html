Data

Data is the collection of raw fcts and figures that can be organized and interpreted into useful information.

Data Types

Data types are the classification of data items. They determine the kind of value a variable can hold and what operations can be performed on it. Common data types include:

  • Integer: Whole numbers without a fractional component.
  • Float: Numbers that contain a decimal point.
  • String: A sequence of characters, often used to represent text.
  • Date: Represents dates and times.
  • Boolean: Represents true or false values.

Database

A database is an organized collection of structured information or data, typically stored electronically in a computer system. It allows for efficient data management, retrieval, and manipulation.

Types of databases

There are several types of databases, each designed for specific use cases:

  • Relational Database: Uses tables to store data and relationships between them (e.g., MySQL, PostgreSQL).
  • NoSQL Database: Designed for unstructured data, often using key-value pairs or document storage (e.g., MongoDB, Cassandra).
  • Graph Database: Focuses on relationships between data points (e.g., Neo4j).
  • Object-Oriented Database: Stores data in objects, similar to object-oriented programming (e.g., db4o).

Database Management System

A Database Management System (DBMS) is software that interacts with databases, allowing users to create, read, update, and delete data. It provides a systematic way to manage data and ensures data integrity and security.
